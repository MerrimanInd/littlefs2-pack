About ESP-Dashboard

An open-source IoT monitoring platform built for makers and professionals.

Built for ESP32

Optimized for the ESP32 family of microcontrollers with support for dual-core processing, Wi-Fi, Bluetooth, and extensive GPIO capabilities.

Custom PCB Design

Optional custom PCB designs available for production deployments with integrated sensor headers, power management, and weatherproof enclosures.

Easy Installation

Flash firmware in minutes with PlatformIO or Arduino IDE. Web-based configuration means no additional software needed after initial setup.

Real-time Dashboard

Beautiful, responsive web dashboard served directly from the ESP32. No cloud services required — all data stays on your local network.

Mesh Networking

Support for ESP-NOW mesh networking to extend coverage across large areas. Nodes automatically relay data to the gateway device.

Open Source

Fully open-source firmware, hardware designs, and documentation. Contribute on GitHub or customize for your specific needs.
